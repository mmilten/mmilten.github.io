Power Supply

A power supply unit (PSU) in a computer converts main AC to low-voltage regulated DC power for all the internal components of a computer. Modern computers universally use switch-mode power supplies. When building a computer one must check if the power supply is enough for all the components of the computer. If there is insufficient power your computer may shutdown, so you have to check carefully. The most power usage of the computer is usually the graphics card, it is all ways a good idea to geat more wattage than a computer is required so this does not occur.

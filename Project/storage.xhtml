Storage

Storage in a computer allows digital data to be saved. Storage devices such as flash devices and hard disks are a fundamental component of a computer which allow the user to save all kinds of information such as video, documents, images, etc. Storage comes in two different types of technology: hard disk drive (HDD) and solid state drive (SSD). SSDs can be further borken down into three types: PCI Express (PCIe), Nonvolatile Memory Express standard (NVMe), and Serial Advanced Technology Attachment (SATA). Each of these drive technology has there performances ranked in performance of read and write speeds.
